Projects

Music Card

I have developed a music card component using HTML, CSS, and JavaScript. The music card displays album artwork, song information, and provides playback controls for users to enjoy their favorite songs.

Train Counter

I have created a train counter application that allows users to track the number of trains passing through a railway station. The application uses JavaScript to increment and decrement the train count and displays the current count in real-time.

Weather Forecast App

I have created a weather forecast application that provides users with real-time weather information and forecasts. The app utilizes APIs to fetch weather data for different locations and displays it in a visually appealing and user-friendly manner. Users can search for specific locations and view detailed weather information, including temperature, humidity, wind speed, and more.

Recipe App

I have developed a recipe application that allows users to search for and view various recipes. The app fetches recipe data from an API and displays the ingredients, instructions, and cooking time. Users can also save their favorite recipes for future reference.

Task Tracker

I have created a task tracker application that helps users manage their daily tasks. The app allows users to add, delete, and update tasks, set due dates, and mark tasks as complete. It provides an organized and efficient way to stay on top of your to-do list.

Movie Recommendation App

I have developed a movie recommendation application that suggests personalized movie recommendations based on user preferences. The app utilizes machine learning algorithms to analyze user preferences and provides tailored movie suggestions for an enhanced movie-watching experience.
